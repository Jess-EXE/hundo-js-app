The Code for HUNDO


                    // Get the values from the page
                    // Start or controller function
                    function getValues() {
                        // Get values from the page
                        let startValue = document.getElementById("startValue").value;
                        let endValue = document.getElementById("endValue").value;
                    
                        // Validate the input
                        // Parse into integers
                        startValue = parseInt(startValue);
                        endValue = parseInt(endValue);
                        let numbers = [];
                    
                        if (Number.isInteger(startValue) && Number.isInteger(endValue)) {
                            // We call generateNumbers
                            numbers = generateNumbers(startValue, endValue);
                            // We call displayNumbers
                            displayNumbers(numbers);
                        } else {
                            alert("You must enter integers")
                        }
                    
                    }
                    
                    // Generate numbers from the startValue to the endValue
                    // Logic function
                    function generateNumbers(sValue, eValue) {
                        let numbers = [];
                    
                        //  Get all numbers from start to end
                        for(let i = sValue; i <= eValue; i++) {
                            numbers.push(i);
                        }
                    
                        return numbers;
                    
                    }
                    
                    // Display the numbers and mark even numbers bold
                    // View function
                    function displayNumbers(numbers) {
                        let templateRows = "";
                    
                        for (let i = 0; i < numbers.length; i++) {
                    
                            let number = numbers[i];
                            let className = "";
                            if (number % 2 == 0) {
                                className = "even";
                            } else {
                                className = "odd";
                            }
                            // The line below contains HTML which does not render correctly with prism
                            // See the source code
                            templateRows += `${number}`;
                        }
                    
                        document.getElementById("results").innerHTML = templateRows;
                        
                    }
getValues()

This function retrieves the values from the DOM where I have the user input the startValue and endValue. After that, I want to make sure that the startValue and endValue are integers, not strings, so I use parseInt to ensure that is the case. I then declare an array named numbers. I then implement some simple conditional logic that will only procede with the code if both startValue and endValue are integers. If they are, then the next two functions are called (explained below). generateNumbers() is assigned to a our numbers array. The numbers array is then passed into the displayNumbers() function as an argument.

generateNumbers()

This function takes two parameters, which will end up being the startValue and endValue I declare the numbers array within this functions scope, and then I have a for loop that iterates through the entirety of the numbers between startValue and endValue inclusive, and add them each to the numbers array. Once the loop is finished, this function returns the array.

displayNumbers()

This function takes the newly populated array numbers as an argument and declares a templateRows variable that I will use for an innerHTML property in this function. I have another for loop that now iterates through each element in the array. Also inside the for loop are two more variables: number, a placeholder for the current iterated element - and className, which will affect our innerHTML class in a moment. If the number is an even, it must be bold, so with more conditional logic I will set the className accordingly (applied by classes in site.CSS). Finally, I countinously add to templateRows. Something to note is that you will not see the HTML here because Prism does not display it correctly. Essentially I make individual rows for the table here. After that, I modify the DOM and add the newly created table for the finsihed results.
